My top restaurants on Beli

Beli has users rank restaurants relative to each other. When a user logs a restaurant, they first categorize it based on whether they 'liked it', if it was 'just ok', or 'did not like it'. Then, Beli has the user compare the restaurant relative to others in the same category by choosing which one they liked more. With the relative rankings, Beli assigns each restaurant a score for the user.
